Это ul и ol теги. Внутри этих тегов обязательно нужны теги li, которые и будут как раз-таки самими элементами списков. Они отличаются тем, что ul - это список маркерами, а ol - это список цифрами и буквами, включая римские цифры.

padding - это внутренний отступ контейнера border - рамка контейнера margin - внешний отступ за контейнером

У CSS есть элементы блочные и строчные. Самый популярный строчный - это тег  Дальше наверное тег  Строчные элементы располагаются рядом то тех пор, пока есть место. Блочные элементы занимают всё доступное место в ширину. **Схлопывание margin** Схлопывание происходит, когда два маржина направленные друг к другу имеют допустим разный размер, один блок 100 пикселей вниз, другой блок ниже находящийся 40 пикселей вверх, в итоге между ними будет расстояние в 100 пикселей. Это и есть схлопывание маржинов. Остановка 3.14

Маржины следует задавать сверху вниз и слева направо. Марджин топ и марджин лефт стараемся не использовать. Правило 1: делай марджины справа и снизу (margin-right, margin-bottom) Правило 2: Элемент не может двигать сам себя, его двигает сосед сверху или слева

Поэтому нужно знать СВОЙСТВА background для изображений:

background-image — свойство для указания пути к фоновому изображению

background-position — свойство для указания месторасположения изображения в контейнере

background-size — свойство для указания размера фонового изображения

background-repeat — свойство которое обязывает повторяться изоброжение на все свободное место контейнера

background-attachment — свойство которое настраивает отображение при прокрутке страницы

 стоит использовать тег для изображений, которые являются частью контента (логотип, иконки на соц.сети, картинки примеров, картинки ссылки)

изображения свойством css background используем для изображений которые являются стилистическим оформлением сайта (картинка фона, картинка оформления)

Есть сокращение для background

Все свойства пишутся через пробел, кроме position и size, их необходимо разделять знаком слеш. Порядок написания значений может быть и производильным, браузер сам определит соответствие свойств и значений.

**Позиционирование** Для того,чтобы размещать элементы друг на друге (по z), для этого у нас есть свойство CSS position. У него есть 5 значений position: static - Элемент ведет себя как обычно position: relative - Элемент можно Позиционировать, относительно своего места. Не вырывается из потока элементов. position: absolute - Вырывает элемент из потока. Позиционируемся от body, если родительскому элементу не задано свойство position:relative position: fixed - Фиксирует элемент при прокрутке в заданном месте. position: sticky - когда докручиваем страницу до элемента, у которого position: sticky, он становится fixed.

Relative — не вырывается, это значит, что элемент будет двигаться относительно соседей, либо относительно родителя.

Absolute — выпал из потока элемента. Остальные элементы ведут себя так, как будто нет этого элемента с absolute. Он позиционируется от тега body, если родителю не задан relative.

Fixed: фиксирует элемент при прокрутке. Он всхлопывается до inline до контента.  
Еще чтобы работали позиционирования не забывай bottom: px; left: px; и right, top.

Z-index: 0 или 1 или -1 и т. д. Работает только с элементами у которых есть какое-нибудь позиционирование например position: relative;
